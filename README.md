# MIRA — Mock Interview and Response Analyzer

An AI-powered mock interview platform. Upload your resume and job description, practice with AI-generated interview questions via voice, and receive structured performance feedback.

## Deployment

Production is live at [miraprep.vercel.app](https://miraprep.vercel.app). Preview deployments are auto-generated for every PR.

See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for environment setup, environment variables, and rollback procedure.
